# Security Policy

## Supported Versions

The following versions of the Palette MCP Server receive security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.** Public disclosure before a fix is available puts all users at risk.

To report a security issue, contact the Spectro Cloud Security team at **security@spectrocloud.com**.

Include the following in your report:

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact (data exposure, privilege escalation, etc.)
- Any proof-of-concept code or screenshots (if applicable)
